Building an Amazon clone:-

Tools and technologies:-

Design: HTML5 and CSS3
Front end: ReactJS and Redux
Database: MongoDB Object Data mapper
Backend: nodeJS,  expressJS
Authentication: JSON web Tokens
Encrypt data: bcrypt
API testing: Postman


We are using the MERN stack for this project (google and fill in holes.)



- Step 1: index.html and style.css (change things here)




- Installed expressJS for Backend
- Installed nodemon for live updation of backend server after updation of backend folder
- Installed axios library to send ajax request to server
- Installed redux and defined reducers to manage state of the items.
- Installed mongoose for managing mongoDB data
